# PocketPantry

PocketPantry is a simple mobile pantry inventory app that helps you keep track of what food you have at home, so you do not accidentally rebuy items you already own.

## Features (MVP)
- Add pantry items with a name, category, and quantity
- View your full inventory in a list
- Search items by name
- Filter / sort items by category
- Edit and delete items
- Saves locally so your inventory is still there after closing the app

## Tech Stack
- Flutter (Dart)
- Local storage: Hive
